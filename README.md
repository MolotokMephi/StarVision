# 🛰️ StarVision — Цифровой двойник группировки кубсатов

**StarVision** — интерактивная 3D-визуализация цифрового двойника группировки российских кубсатов «Сфера-КС». Проект создан для Национального центра космических компетенций.

![Status](https://img.shields.io/badge/Статус-В_разработке-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## Особенности

- **Реалистичная орбитальная механика** — законы Кеплера, J2-прецессия, уравнение Кеплера (метод Ньютона)
- **3D-визуализация** на Three.js с текстурами NASA Blue Marble + ночные огни
- **Walker-δ группировка** — параметризация: кол-во плоскостей, спутников, высота, наклонение, эксцентриситет
- **Телеметрия в реальном времени** — период, скорость, покрытие, энергобюджет, радиолиния
- **Наземные станции** — российские НС с корректным вращением вместе с Землёй
- **Карта покрытия 2D** — проекция Plate Carrée с зонами покрытия
- **AI-ассистент** — управление визуализацией голосовыми командами на русском языке
- **Подготовка к реальным данным** — модули парсинга TLE, каталог российских кубсатов

## Технологический стек

| Компонент | Технология |
|-----------|-----------|
| Frontend | Vanilla JS (ES Modules), Three.js r128 |
| Backend | Node.js, Express |
| 3D-рендеринг | Three.js + custom GLSL шейдеры |
| Орбитальная механика | Собственная реализация (Кеплер, SGP4-совместимый) |
| Данные | CelesTrak, NORAD TLE, каталог российских кубсатов |
| AI | Встроенный NLP-модуль + API для внешней нейросети |
| Дизайн | Liquid Glass UI, Mission Control стиль |

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск (backend + frontend)
npm run dev

# Только frontend (без backend)
npm run frontend

# Только backend
npm run backend
```

Приложение будет доступно по адресу: `http://localhost:3000`

## Орбитальная механика

Проект реализует следующие физические модели:

### Кеплеровы элементы
- **Большая полуось (a)** = R⊕ + h
- **Эксцентриситет (e)** — форма эллипса
- **Наклонение (i)** — угол к экватору
- **RAAN (Ω)** — долгота восходящего узла
- **Аргумент перицентра (ω)**
- **Истинная аномалия (ν)**

### Ключевые формулы
| Формула | Описание |
|---------|----------|
| `T = 2π√(a³/μ)` | Третий закон Кеплера (период) |
| `V = √(μ/a)` | Круговая орбитальная скорость |
| `M = E - e·sin(E)` | Уравнение Кеплера |
| `dΩ/dt = -3/2·n·J2·(R⊕/p)²·cos(i)` | J2 прецессия RAAN |
| `dω/dt = 3/2·n·J2·(R⊕/p)²·(2-5/2·sin²i)` | J2 прецессия аргумента перицентра |
| `GMST(t) = 2π(0.779... + 1.00274...·du)` | Гринвичское звёздное время |

### Константы (ГОСТ Р 25645.166-2004, NASA GSFC)
- μ = 3.986004418 × 10¹⁴ м³/с² — гравитационный параметр Земли
- R⊕ = 6371.0 км — средний радиус Земли
- J2 = 1.08263 × 10⁻³ — коэффициент J2 геоида

## AI-ассистент

Встроенный AI-ассистент понимает команды на русском языке:

```
«покажи спутник СК-0301»     → Выбирает спутник
«переключи камеру на СК-02»  → Наводит камеру
«ускорь время»               → Устанавливает ×100
«скорость ×1000»             → Произвольная скорость
«покажи покрытие»            → Включает визуализацию покрытия
«высота 600»                 → Изменяет высоту орбиты
«статус»                     → Показывает текущую телеметрию
«помощь»                     → Список всех команд
```
