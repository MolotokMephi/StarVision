# Дорожная карта StarVision

## Базовый функционал (MVP)

- [x] 3D-визуализация Земли (текстуры NASA Blue Marble, ночные огни, атмосфера)
- [x] Walker-δ генерация группировки кубсатов
- [x] Орбитальная механика (Кеплер, J2-прецессия, GMST)
- [x] Параметризация: количество плоскостей, спутников, высота, наклонение, эксцентриситет
- [x] Телеметрия в реальном времени (период, скорость, покрытие)
- [x] Наземные станции России с корректным вращением
- [x] Управление временем (пауза, скорость ×1–×1000)
- [x] 2D-карта покрытия (Plate Carrée проекция)
- [x] Энергобюджет КА (генерация, потребление, АКБ)
- [x] Бюджет радиолинии (UHF, FSPL, SNR)
- [x] Разделение на backend и frontend
- [x] AI-ассистент — базовая структура
